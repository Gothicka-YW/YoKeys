# YoKeys Privacy Policy

Effective date: March 30, 2026

YoKeys is a simple web tool that converts supported YoWorld home-link formats into a canonical format.

## Summary

- YoKeys does not require account login.
- YoKeys does not sell personal information.
- YoKeys does not run ads or trackers.
- YoKeys processes your input in your browser.

## Information We Handle

YoKeys may handle the text you type or paste into the input field, such as:

- A numeric home ID
- A supported YoWorld or Facebook-format URL containing a home ID

This input is used only to generate the canonical output link.

## Local Storage

If you enable the "Remember my last Home ID" option, YoKeys stores the last parsed Home ID in your browser's localStorage so it can be prefilled later.

- Stored locally on your device/browser only
- Not transmitted to a server by YoKeys
- You can clear it at any time by unchecking the remember option and/or clearing browser site data

## Data Collection and Sharing

YoKeys does not intentionally collect, store, or share personal data on external servers.

YoKeys does not send your entered Home ID to a backend API for processing.

## Third-Party Services

YoKeys is hosted on GitHub Pages. GitHub may process technical request data (such as IP address, logs, and basic telemetry) as part of web hosting. See GitHub's policies for details:

- https://docs.github.com/site-policy/privacy-policies/github-general-privacy-statement

YoKeys also loads web fonts from Google Fonts, which may involve requests to Google-hosted font endpoints.

## Children's Privacy

YoKeys is a general utility tool and is not directed to children under 13.

## Changes to This Policy

This policy may be updated if YoKeys features or hosting behavior changes. The effective date at the top will be updated when changes are made.
